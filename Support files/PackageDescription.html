EasyGoSquared Pro Component and Plugin

EasyGoSquared Pro is a combination of standard Joomla plugins and a component that adds the GoSquared Analytics scripts to your websites pages, while providing direct access to your GoSquared Dashboard inside Joomla's Admin area.

Usage

There's only a few things to do to get up and running.

  1. open the content plugin and set it up
    1. add the GoSquared Token for your website to the Content Plugin
    2. choose whether to have GoSquared analytics for the Admin users
    3. specify what user attributes you want to track
    4. specify any user profile values you want to track
    5. enable the plugins
  2. open the EasyGoSquared component
    1. click on the "Options" button
    2. add your GoSquared API key
    3. add your GoSquared website token
    4. Choose the default

Don't forget to enable and configure the plugin in the Plug-in Manager.
